Welcome to the Bookstore API documentation! This is a RESTful API built with Node.js, Express, and MongoDB to manage information about books and authors in a bookstore.

## Overview

Bookstore API provides endpoints to perform CRUD operations (Create, Read, Update, Delete) on the following resources:

- **Authors**: Manage information about authors, including name, birth year, and their list of books
- **Books**: Manage information about books, including book title, publication date, genres, and author

## Key Features

- RESTful API with clear endpoints following RESTful standards
- MongoDB for data storage
- Full CRUD operation support
- Complete API documentation with Zudoku
